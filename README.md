# GRCodingExam

To clone this repo:

`git clone https://github.com/dapuck/GRCodingExam.git`

Install dependencies:

`npm i`

Run tests

`npm test`

Command line

```
Usage: node index.js <file1> <file2> <file3>

file1     Pipe delimited file
file2     Comma delimited file
file3     Space delimited file
```

Start server

`npm start`
